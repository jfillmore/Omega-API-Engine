Overview

Omega has two primary components.

  • PHP Framework for hosting RESTful APIs.
  • JavaScript libraries for GUI building.

The PHP server uses Nginx and PHP-FastCGI. It includes configuration scripts for upstart/initctl for the services "nginx" and "omega-php". By default nginx runs HTTPS on port 5800 (and optionally 5858 for insecure). PHP-FastCGI runs as a daemon on port 5900.

Requirements

  • PHP 5.2+ with FastCGI support and support for the following:
    • JSON
    • MySQL/MySQLi
    • Curl
  • nginx 0.8.x+
  • python 2.3+ w/ JSON or SimpleJSON modules for command-line tools.

Supported Distributions

  • Ubuntu 10+
  • RHEL 4+/CentOS 5+
  • Other distribution supported with manual configuration.
